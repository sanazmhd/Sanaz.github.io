SQL Learning

SQL is a standard language used for storing, manipulating, and retrieving data in databases. In this SQL tutorial, we will explore how to use SQL in various database systems including MySQL, SQL Server, MS Access, Oracle, Sybase, Informix, PostgreSQL, and others ..

When creating a website to display data from a database, you'll require the following:

  1. An RDBMS (Relational Database Management System) database system (e.g., MS Access, SQL Server, MySQL). RDBMS is the basis for SQL and for all modern database systems such as MS SQL Server, IBM DB2, Oracle, MySQL, and Microsoft Access. The data in RDBMS is stored in database objects called tables. A table is a collection of related data entries and it consists of columns and rows.
  2. A server-side scripting language such as PHP or ASP.
  3. Utilization of SQL for retrieving the desired data.
  4. Incorporation of HTML/CSS for page styling.
